【交互说明】
═══════════════════════════════════════════════════════════════════════════════════════════════════════════

1. 悬浮窗交互：
   - 输入框：支持粘贴链接或拖拽链接
   - 采集按钮：点击后触发采集流程
   - 文章卡片：点击[查看]进入详情，点击星标切换收藏状态
   - 窗口：可拖动，置顶显示，支持自动隐藏

2. 主窗口交互：
   - 搜索栏：输入关键词实时搜索（防抖300ms）
   - 筛选器：下拉选择账号、标签、时间范围
   - 文章列表：多选框批量选择，卡片点击查看详情
   - 批量操作：删除、添加标签、导出、收藏
   - 侧边栏：快速切换不同功能模块

3. 详情页交互：
   - 返回按钮：返回列表
   - 收藏/标签/导出按钮：快捷操作
   - 笔记编辑：点击[✏️ 编辑]进入编辑模式
   - 导出格式：选择Markdown/HTML/PDF

4. 标签管理交互：
   - 新建标签：设置名称和颜色
   - 编辑标签：修改名称、颜色、合并标签
   - 删除标签：删除标签并解除文章关联
   - 文章列表：点击文章标题查看详情

5. 数据统计交互：
   - 时间范围：切换不同统计周期
   - 图表：鼠标悬停显示详细数据
   - 词云：点击关键词筛选相关文章
   - 导出报告：生成可视化报告文件

【状态提示说明】
═══════════════════════════════════════════════════════════════════════════════════════════════════════════

采集状态：
  - 🟢 + "已采集" = 采集成功
  - 🟡 + "文章已存在" = 重复采集警告
  - 🔵 + 普通状态 = 待处理
  - ⊗ + "抓取失败" = 采集失败

收藏状态：
  - ⭐ = 已收藏（金色星标）
  - ☆ = 未收藏（空心星标）

操作反馈：
  - ✅ = 操作成功
  - ⚠️ = 需要注意
  - ❌ = 操作失败
  - ⓘ = 提示信息

【快捷键说明】
═══════════════════════════════════════════════════════════════════════════════════════════════════════════

全局快捷键：
  - Ctrl+Shift+A  = 显示/隐藏悬浮窗
  - Ctrl+F        = 聚焦搜索框
  - Ctrl+N        = 打开采集窗口
  - Ctrl+S        = 保存当前笔记

列表操作：
  - ↑/↓          = 上下选择文章
  - Enter        = 打开文章详情
  - Space        = 切换收藏状态
  - Delete       = 删除选中文章

详情页：
  - Esc          = 返回列表
  - Ctrl+E       = 编辑笔记
  - Ctrl+D       = 导出文章
