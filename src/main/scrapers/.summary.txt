================================
微信公众号文章抓取模块
================================

📁 文件结构：
-----------
wechat.ts          (315行) - 核心抓取器实现
types.ts           (39行)  - 类型定义
index.ts           (24行)  - 模块导出
example.ts         (167行) - 使用示例
test.ts            (90行)  - 测试脚本
README.md          (7.3K)  - 使用文档
ARCHITECTURE.md    (7.1K)  - 架构文档
QUICKSTART.md      (5.4K)  - 快速开始

总计：635行代码，3份文档

✅ 核心功能：
-----------
1. 抓取微信公众号文章（标题、作者、时间、封面、摘要）
2. 单篇和批量抓取支持
3. 自动提取文章摘要（前200字）

🛡️ 防护机制：
-----------
1. 随机 User-Agent（6个常用浏览器）
2. 请求间隔控制（默认2秒）
3. 超时和重试（默认3次，指数退避）
4. 请求超时（默认10秒）

❌ 错误处理：
-----------
1. URL格式验证
2. 网络异常捕获
3. HTML解析失败处理
4. 详细错误信息返回

📦 技术栈：
-----------
- axios: HTTP请求
- cheerio: HTML解析
- TypeScript: 类型安全

🚀 快速使用：
-----------
import { scrapeWechatArticle } from './scrapers';

const result = await scrapeWechatArticle('https://mp.weixin.qq.com/s/xxx');
if (result.success) {
  console.log(result.data.title);
}

📝 配置选项：
-----------
{
  maxRetries: 3,        // 最大重试次数
  retryDelay: 2000,     // 重试延迟（毫秒）
  timeout: 10000,       // 请求超时（毫秒）
  requestInterval: 2000 // 请求间隔（毫秒）
}
